# Developer
給開發者的參考文件

### 架構
* Firebase Authentication - Google 登入
* Firebase Firestore - 資料庫
* Firebase Functions - 後端
* Firebase Hosting - 前端

### 環境設定

**Node.js**
```sh
# 確認 node 正確安裝
node -v

# 確認 npm 正確安裝
npm -v
```

**Firebase**
```sh
# 安裝 firebase 開發工具
npm install -g firebase-tools

# 登入 firebase
firebase login
```
